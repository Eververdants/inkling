
## Purpose

This directory holds the **output** of the `/inkling` brainstorming skill. When you run `/inkling` in a project, the final 6-section proposal is written here.

## `docs/ideas/`

Each run of the 5-stage conversation produces one markdown proposal:

`docs/ideas/YYYY-MM-DD-<slug>-idea.md`

- `<slug>` is a lowercase kebab-case name derived from the problem statement (2–4 words).
- Each file contains: One-liner → Problem → Target User → MVP Features → Why You, Why Now → Risks & Mitigations.
- This directory is **gitignored** in this skill repo — the proposals belong to the project where you run `/inkling`, not to the skill itself.

## Note for contributors

`docs/` exists at the skill repo root as a **reference layout** — it shows where proposals land when you use the skill inside a project. The directory is empty by design; actual proposals are generated at runtime in the calling project.
